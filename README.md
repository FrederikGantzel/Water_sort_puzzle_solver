# Water Sort Puzzle Solver
A program used to solve levels of the mobile game Water Sort Puzzle, if you feel like you're stuck.

## Installation
Download the "Water_sort_puzzle_solver" folder and put it on your desktop or wherever else feels best for you. The program is written with Processing, and thus it is easily run using this application. Processing can be downloaded here: https://processing.org/download

When you have Processing installed, simply open the Snake_Game folder in Processing and press the "Play" button.

## Usage
Use your mouse to click on the vials and change the colors, to align the vials with what you see on your phone screen. Once you have matched the color of the vials in the program with the color of the vials in the game, simply click "Start" to star the solving-algorithm. Note that you can change between 11-vial and 14-vial levels with the toggle at the top right of the program UI. Once a solution is found, it will be printed in the Processing console. You can pull up a pre-programmed test-level by pressing the "t" key on your keyboard. The test-level will be different depending on whether "14 vial leve" is toggled or not.

## Limitations
Some levels, especially the larger 14-vial levels, may take a very long time to complete (possibly multiple days). The pre programmed 14-vial level for example is an unsolvable level, and the algorithm was not able to explore all possible ways to play this level, even after being left running overnight. This is because the solving algorithm is currently very unoptimized, but I hope to cut the time down to more manageable levels eventually.
